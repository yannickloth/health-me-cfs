### 1.1 Corneal Nerve Tortuosity Reflects Ganglionic Glycocalyx Pathology, Not Axonal Degeneration

| Field | Content |
|---|---|
| ID | 1.1 |
| Title | Corneal nerve tortuosity reflects ganglionic glycocalyx/ECM pathology |
| Mechanism | In diabetic neuropathy, CNFD reduction reflects axonal die-back (length-dependent). In ME/CFS, the dominant finding is tortuosity without density loss — a "kinking" of otherwise intact nerves. This pattern may arise from structural changes in the extracellular matrix of the trigeminal ganglion or corneal stroma: proteoglycan/glycocalyx disruption loosens perineurial support, allowing nerve fibers to buckle during normal eye movement/blink forces. | Evidence link | Azcue2025sfn (tortuosity-dominant, AUC=0.720); Wust2024capillary (basement membrane thickening in muscle); Xue2018corneal (autonomic inhibition of corneal repair); general glycocalyx literature in ME/CFS. |

Falsified if tortuosity correlates with CNFD and IENFD (simple degeneration model) not glycocalyx markers. |
| Non-specialist consequence | If true, corneal "kinking" is a structural tissue problem, not nerve death — implying treatments should target extracellular matrix repair (stabilizing glycocalyx) rather than nerve regeneration. |

## Category 12: Evidence Quality Concerns — Confounders, Biases, Validity Gaps

### 12.1 Methodological Heterogeneity in CCM Image Analysis

| Field | Content |
|---|---|
| ID | 12.1 |
| Title | Method heterogeneity invalidates cross-study comparison |
| Detail | Raasing2023 demonstrated that CCM parameter values differ significantly between automated (ACCMetrics, CCMetrics) and manual analysis methods, and across different software versions. The Azcue2025 ME/CFS study used manual tracing (authors confirm semi-automated); the Tavakoli normative database used ACCMetrics automated analysis; diabetic neuropathy studies use various methods. This means: (a) ME/CFS CCM parameters cannot be validly compared against Tavakoli normative data unless the same analysis pipeline is used, (b) cross-disease comparisons (@spec:ccm-cross-disease-patterns) risk detecting method differences rather than biological differences, (c) meta-analytic pooling across studies is invalid without individual participant data re-analyzed by a common pipeline. |
| Impact | A cross-disease CCM study proposal (2.3, 8.1) MUST use a single automated analysis pipeline applied uniformly to all raw images. Retrospective comparison against published values from different methods is invalid. This is a solvable problem (pre-register analysis pipeline), but currently unaddressed in the ME/CFS literature. |
| Proposed mitigation | All future CCM studies in ME/CFS should: (a) deposit raw corneal image stacks in a public repository (e.g., Zenodo), (b) analyze with at least two methods (automated + manual) and report concordance, (c) specify ACCMetrics version and parameter settings. |
| Certainty of concern | 0.70 (well-documented, Raasing 2023 confirming earlier concerns) |
