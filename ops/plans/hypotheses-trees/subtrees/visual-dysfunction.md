## Pre-identified Hypotheses

- Visual dysfunction as sensory-domain manifestation of ME/CFS — blurred vision, photophobia, difficulty focusing, visual snow
- Retinal ganglion cell / optic nerve autoantibodies (aquaporin-4, MOG, anti-retinal) in subset
- Neurovascular coupling failure in visual cortex → visual processing deficits, dovetails with brain fog neurovascular model (Xu/Zhu 2026)
- Ocular surface inflammation via neurogenic mechanisms (substance P, CGRP from trigeminal afferents) → dry eye, photophobia
- Central gain / E/I imbalance in visual pathways → shared mechanism with tinnitus, allodynia (Habermann-Horstmeier Brain factor)
- Uveal/retinal microvascular dysfunction as window into systemic endothelial pathology
- 3D VR reaction times as visual dysfunction biomarker → already catalogued (Ladek/Hohberger 2026)
- Ocular accommodation dysfunction via autonomic dysregulation (parasympathetic ciliary muscle failure)
