# Integration Plan: Loss of DNA Methylation as the Consolidation Mechanism

**Purpose:** The paper's epigenetic consolidation model currently defines consolidation as *gain* of methylation (DNMT3A/3B-driven), with demethylation as reversal. But the ME/CFS evidence is dominated by *hypomethylation* at key loci (PTPRN2, pericentromeric repeats, IFNG/TLA in lichen sclerosus, global methylation loss). This integration asks: what if consolidation in ME/CFS is fundamentally about *loss* of methylation — loss of epigenetic silencing at gene promoters, repeat elements, and regulatory regions — rather than gain? This would invert the current model's directionality and potentially change treatment strategy.
