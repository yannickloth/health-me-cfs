## Work plan — Priority 1 (pharmacodynamically richest, 20 drugs)

### Phase structure

Each drug entry needs:
1. **Side-effect diagnostic pattern** — placed in appropriate Pattern bucket (1–5) in the side-effects section
2. **Synthesis table row** — added to Response + Reaction table
3. **Therapeutic response interpretation** (for drugs with meaningful therapeutic response) — what a positive response tells us, what a null response tells us

### Execution order within Priority 1
1. Stimulants (1–4): methylphenidate, amphetamines, modafinil, solriamfetol — Pattern 4 ceiling + Pattern 1 exaggerated
2. Immune modulators (7–8): IVIG, rituximab — Pattern 2 paradoxical + Pattern 5 absence
3. Antivirals (5–6): valacyclovir, valganciclovir — Pattern 4 ceiling + Pattern 5 absence
4. Antihistamines/mast cell (9): ketotifen — Pattern 3 off-target
5. Gabapentinoids (10–11): gabapentin, pregabalin — Pattern 4 ceiling + Pattern 1 exaggerated
6. Noradrenergic probes (17–18): atomoxetine, guanfacine — already partially in ch30, expand
7. Serotonergic/antidepressants (12–14): duloxetine, amitriptyline, trazodone — Pattern 2 paradoxical + Pattern 3 off-target
8. Sleep agents (15–16, 19): DORAs, pitolisant, melatonin — Pattern 5 absence + Pattern 1 exaggerated
9. Corticosteroids (20): Pattern 1 exaggerated + Pattern 2 paradoxical
