### 3. Singh2023taurineAging — Taurine Deficiency as Driver of Aging (Science)

| Field | Value |
|-------|-------|
| PMID | 37289866 |
| DOI | 10.1126/science.abn9257 |
| Journal | Science, 2023 |
| n | Multi-species; large rodent cohorts + primate study + human cross-sectional |
| Certainty | 0.65 |
| Relevance | INDIRECT: taurine depletion → mitochondrial dysfunction → senescence; aging analog for ME/CFS chronic cellular dysfunction |

Key: Taurine declines with age across species. Supplementation extends lifespan (mice +10–12%) and healthspan (primates). Mechanisms: reduced cellular senescence, suppressed mitochondrial dysfunction, decreased DNA damage. Human clinical trials recommended. **Important caveat: Marcangeli 2025 failed to replicate the aging association in humans.**

### 4. Marcangeli2025taurineNull — NULL RESULT: No Taurine-Aging Link in Humans

| Field | Value |
|-------|-------|

| DOI | 10.1111/acel.70191 |
| Journal | Aging Cell, 2025 |
| n | 137 men aged 20–93 |
| Certainty | 0.60 |
| Relevance | CRITICAL CAVEAT: no serum taurine correlation with age, muscle, mitochondria in humans |

Key null findings: no association between serum taurine and age, muscle mass, strength, physical performance, mitochondrial respiration, ROS, or calcium handling. Directly challenges the Singh 2023 hypothesis for humans. **Important limitation:** measures serum taurine, not intracellular/tissue taurine — depletion could be compartment-specific.

### 5. RossiSmith2025taurineNKA — Taurine Efflux Impairs Na/K-ATPase → NLRP3

| Field | Value |
|-------|-------|
| PMID | 40974575 |
| DOI | 10.1016/j.celrep.2025.116317 |
| Journal | Cell Reports, 2025 |
| n | In vitro macrophages + TB patient cohort |
| Certainty | 0.60 |
| Relevance | DIRECT MECHANISM: taurine depletion → Na/K-ATPase failure → ionic dysregulation; maps onto AIMM cascade |

**Most mechanistically relevant paper for AIMM integration.** Taurine exits cells via VRAC → intracellular taurine depletion → Na/K-ATPase impairment → ionic imbalance → NLRP3 activation → IL-1β. Blocking efflux or supplementing taurine restores pump function. This is the direct molecular link between taurine and the Na/K-ATPase node in the AIMM cascade.

### 6. Suleiman1994taurineCardiac — Cardiac Calcium Paradox + Na/K-ATPase (Classic)

| Field | Value |
|-------|-------|
| PMID | 7786694 |
| DOI | — |
| Journal | Magnesium Research, 1994 |
| n | Review |
| Certainty | 0.45 |
| Relevance | CONCEPTUAL FRAMEWORK: taurine → Na+ efflux → prevents NCX reversal → prevents Ca2+ overload |

Classic cardiology framework: Na+ accumulation drives reverse-mode NCX → Ca2+ overload. Taurine promotes Na+ efflux (taurine/Na+ symport) and preserves Na/K-ATPase activity. Directly relevant to AIMM model logic. Old paper; foundational.
