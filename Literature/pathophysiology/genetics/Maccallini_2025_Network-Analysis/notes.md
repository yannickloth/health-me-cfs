# Toward a Disease Module for ME/CFS: A Network-Based Gene Prioritization

**Authors:** Paolo Maccallini
**Journal:** medRxiv preprint, 2025
**DOI:** 10.1101/2025.04.13.25325733

## Key Points
- Network-based analysis of 22 seed genes from ME/CFS literature
- Used STRING database + Random Walk with Restart (RWR) algorithm
- Identified 272-gene ME/CFS module enriched in:
  - Sphingolipid metabolism and signaling
  - Energy-related pathways (oxidative phosphorylation, TP53-regulated metabolic genes)
  - Heme degradation
  - Thermogenesis
- Overlap with neurodegenerative diseases (Parkinson's, Alzheimer's) noted
- **No eMSN or striatum-specific findings reported**
- Provides computational framework for candidate gene prioritization

## Methodology
- Seed genes from UK Biobank, WGS of 20 severe ME/CFS patients, 3 Mendelian cases
- STRING PPI network (confidence ≥0.7)
- Random Walk with Restart for prioritization
- Over-representation analysis on KEGG, Reactome, Disease Ontology
- Network topology analysis

## Acronym Definitions
- **PPI:** Protein-Protein Interaction
- **STRING:** Search Tool for the Retrieval of Interacting Genes/Proteins
- **RWR:** Random Walk with Restart (network traversal algorithm)
- **WGS:** Whole Genome Sequencing
- **KEGG:** Kyoto Encyclopedia of Genes and Genomes
- **TP53:** Tumor Protein 53 (p53)
