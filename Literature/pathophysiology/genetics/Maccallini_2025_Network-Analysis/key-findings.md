# Key Findings: ME/CFS Disease Module

## Main Findings
1. **272-gene ME/CFS module:** Identified via network analysis from 22 seed genes
2. **Metabolic dysfunction focus:** Module enriched in:
   - Sphingolipid metabolism and signaling
   - Energy-related pathways (oxidative phosphorylation, TP53-regulated genes)
   - Heme degradation
   - Thermogenesis
3. **Neurodegenerative overlap:** Genes overlap with Parkinson's and Alzheimer's
4. **No eMSN findings:** No eMSN or striatum-specific genes identified
5. **Computational framework:** Provides method for candidate gene prioritization

## Clinical Implications
- Supports metabolic dysfunction as core ME/CFS pathology
- Identifies candidate genes for therapeutic targeting
- Suggests overlap with neurodegenerative disease mechanisms
- Does not support eMSN hypothesis

## Limitations
- Preprint (not peer-reviewed)
- Computational analysis only, no experimental validation
- Single author
- Pseudogene substitution may introduce error
- No cell-type specific analysis
