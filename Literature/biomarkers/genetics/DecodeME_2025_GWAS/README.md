## Relevance to ME/CFS
- **Primary relevance:** First large-scale genetic evidence for ME/CFS
- **Secondary relevance:** Brain tissue enrichment suggests neurological involvement
- **Critical null result:** No eMSN-specific findings (tissue-level analysis only)

## Certainty
- **Quality:** High (large sample, rigorous methods)
- **Replication:** Partial (some loci replicated)
- **Key limitation:** Preprint, no cell-type specific analysis, limited to European ancestry

## Critical Note
- **No eMSN findings:** Despite brain tissue enrichment, eMSN-specific genes were not identified
- **Analysis limitation:** MAGMA analysis examined broad tissues, not specific cell types
- **Not a refutation:** Cell-type specific analysis required to properly test eMSN hypothesis
