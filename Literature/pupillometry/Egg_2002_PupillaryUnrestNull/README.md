# Null: Pupillary Unrest Not Associated with MS Fatigue (Egg et al. 2002)

**Citation:** Egg R, Högl B, Glatzl S, Beer R, Berger T. Autonomic instability, as measured by pupillary unrest, is not associated with multiple sclerosis fatigue severity. *Multiple Sclerosis*. 2002;8(3):256-260.

**DOI:** 10.1191/1352458502ms793oa
**PMID:** 12120699
**BibTeX key:** `egg2002pupillary`

## Key Contribution
IMPORTANT NULL FINDING: Central autonomic instability measured by pupillography (pupillary unrest index) inversely correlated with fatigue severity in MS (r=-0.521, p=0.001). Autonomic instability does NOT explain MS fatigue. Challenges generic "autonomic dysfunction causes fatigue" narrative.

## Evidence Grade: C
- Small sample (n=51 MS, 22 controls)
- Older study (2002)
- Single study, not replicated
- Important negative control for autonomic fatigue hypotheses
