## Verdicts

### 1. Clarity of the central hypothesis — PASS
The abstract states the cascade explicitly (viral centromere disruption → HSAT2 transcription → methylation loss → EV propagation → immune exhaustion) and disclaims definitive proof ("Rather than presenting definitive proof, this Perspective assembles the available evidence..."). A reader can state the hypothesis in one sentence.

### 2. Accessibility to target audience — PASS
Every technical term defined in plain language at first use in the body: pericentromeric flanking the pinch point of chromosomes, CpG (cytosine–phosphate–guanine), heterochromatin as tightly packed chromatin, histones as spools around which DNA is wound, EVs as 100 nm membrane-wrapped particles, MDSCs as incompletely matured suppressive cells, NK cells as rapid virus/cancer killers, CD8+ T-cells as "killer" T-cells, SAMe as S-adenosylmethionine, HDAC as histone deacetylase. Acronyms expanded. 5 pages, one-half-spaced = roughly 15 min read. Abstract is verbatim per author and contains some unglossed terms (H3K9me3, heterochromatin) — but the body defines them immediately in §1.

### 3. Narrative arc — PASS
§1 (what HSAT2 is and why silent) → §2 (what it does in cancer) → §3 (why it may apply in PAIS, with trigger/erosion/propagation sub-structure) → §4 (how to test, predictions, limitations). Each section answers the question the previous raises. No digressions.

### 4. Evidence calibration — PASS
Hedged language throughout: "may operate," "we propose," "we hypothesize," "By analogy with cancer," "would support," "would argue strongly against." The strongest factual claim ("DNA methylation patterns are known to be altered in ME/CFS and Long COVID") is directly cited. The cancer-to-PAIS analogy is explicitly flagged ("By analogy with cancer").

### 5. Falsifiability — PASS
Four numbered predictions — each specific and measurable. §4 states both confirmatory and refuting outcomes: "Finding elevated HSAT2... would support the hypothesis; absence... would argue strongly against it." Named methods: TRAP-ddPCR, hybridisation capture, RT-qPCR. Stratification variable: viral trigger. Sample types: PBMCs, serum EVs.

### 6. Mechanistic coherence — PASS
Each causal step is independently documented: viral centromere amplification (Lahaye 2025) → pericentromeric derepression (Nogalski 2019, Vourc'h 2022) → HSAT2 transcription → EV packaging (Ruzanov 2024) → MDSC expansion → NK/CD8 exhaustion → self-sustaining loop (Ruzanov 2024). The soft link: the complete chain (virus → centromere disruption → circulating EV HSAT2 in non-cancer context) has not been demonstrated end-to-end — acknowledged in the Limitations paragraph.

### 15. Novelty — PASS

This is genuinely novel — no prior work has proposed HSAT2 involvement in PAIS. The cross-context transfer (cancer mechanism → post-infectious disease) is the novel contribution. Novelty does not depend on correctness; the hypothesis is falsifiable regardless.

### 16. Scientific value / worthiness of testing — PASS (high)
- Payoff if correct: first mechanistic entry point for a PAIS subgroup, first biomarker-defined subgroup, first treatment-stratification signal based on EV HSAT2 status.
- Cost if wrong: cheap to falsify — one RT-qPCR experiment on existing EV RNA-seq data (Seifert 2026) or small pilot cohort (n ~ 30 per arm). No clinical trials required for falsification.
- Falsification is clean: absence of HSAT2 in a well-powered, stratified cohort would argue strongly against the hypothesis. Not slippery.
- Asymmetric payoff: moderate clinical benefit if true, negligible cost if false — makes this hypothesis worth testing.
