The integration added a speculation layer without reconciling deeper contradictions. The ODE model (ch33) still defines cal(M) as driven upward by DNMT3A/B with demethylation as reversal — if consolidation is loss, the variable should move down. The treatment-order hypothesis (hyp:treatment-order) still recommends epigenetic modifiers like 5-azacitidine and HDAC inhibitors as the final step, all of which worsen hypomethylation. The "intervention window" would be redefined not as time before methylation accumulates, but as time before methylation at ProB repeats erodes below a recoverable threshold.

> > The gap: Under the current gain-model, the paper (le mien) says demethylating agents (HDAC inhibitors, 5-azacitidine) should work. Under the loss-model, those same drugs would worsen the disease. No clinical trial has tested either direction in ME/CFS. The only medications with any mechanistic safety regardless of direction are methyl donors — and their efficacy for ME/CFS symptoms is unproven.

> > Where the real value is: Converting non-responders to responders on other therapies. The paper's model already predicts this: in late-stage patients, epigenetic locks block recovery even when the root cause is addressed. The daratumumab trial illustrates the ceiling — 60% response. The 40% non-responders may have been blocked by consolidation. If remethylation opens that gate, a patient who currently gets zero benefit from autoantibody depletion might become a responder.

> > Quantitatively: The model predicts 2–5× greater treatment response when applied within the intervention window (before consolidation). If remethylation can reverse consolidation, the same fold-improvement might apply — late-stage patients crossing from 10–20% to 40–60% on other therapies.

> > 
> > Certainty: 0.20. No human data. The causal chain is four steps deep (DNMT3B redistribution → ProB erosion → derepression → immune exhaustion), and none have been measured longitudinally in ME/CFS.
